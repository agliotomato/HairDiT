# T2 Sanity-Test v2 — Sparsity Ladder (raw colored sketch, recolor X)

> v1과 동일 setup, 단 sketch는 GT-recolor 적용 X — 원본 dataset의 다색(알록달록) stroke 그대로 사용.

---

## 1. 실험 개요

| 항목 | 내용 |
|------|------|
| 목적 | sparsity가 증가할 때 MatteCNN의 기여 — sparse에서 CNN 이득 확대 여부 검증 (raw color 조건) |
| 비교 모델 | **mcs1 ~ mcs6 전체 6구성** (핵심 쌍: Ours vs Raw-only) |
| 입력 고정 | **raw colored sketch (3 sparsity, recolor X)** + GT matte + face_B |
| 변수 | **스트로크 수 sparsity 3단계** (dense / mid / sparse) |
| 타겟 조건 | **B** — HairMapper bald + 원본 장면 |
| 이미지 | CM_1005 / CM_1033 / CM_1067 / CM_1068 / CM_1077 / CM_1082 / CM_1101 / CM_1106 (8장) |
| 측정 지표 | **Edge IoU (vs sketch, matte 영역)** + **hair 영역 Chroma 평균** |
| seed / steps | T2_v1과 동일 |

> **예측 (design.md L104)**: sparse할수록 CNN 이득 확대 (mcs1 > mcs5). Raw-only(mcs5) muddy.

### Sparsity 정의

| 단계 | keep ratio | 비고 |
|---|---|---|
| **dense**  | 100% (원본 색 모두 keep) | 원본 sketch 그대로 |
| **mid**    | 50% | random color subsample, seed=42 |
| **sparse** | 20% | random color subsample, seed=42 |

> **v1과의 차이**: sketch는 **GT-recolor 적용 X**. 원본 dataset(SketchHairSalon paired) 의 다색 stroke 그대로.

## 3. 종합 측정 (8 stems 평균)

### Edge IoU

| sparsity | mcs1 (Ours) | mcs2 (Ours+Gate) | mcs3 (Sketch-only) | mcs4 (Sketch-only+Gate) | mcs5 (Raw-only) | mcs6 (Matte-CNN-only) |
|---|:---:|:---:|:---:|:---:|:---:|:---:|
| dense | 0.0460 | 0.0457 | **0.0462** | 0.0446 | 0.0450 | 0.0442 |
| mid | **0.0275** | 0.0260 | 0.0271 | 0.0261 | 0.0257 | 0.0253 |
| sparse | 0.0096 | **0.0127** | 0.0087 | 0.0090 | 0.0086 | 0.0117 |

### Chroma 평균 (Lab √(a²+b²), hair 영역, ↑ 자연 / ↓ muddy)

| sparsity | mcs1 (Ours) | mcs2 (Ours+Gate) | mcs3 (Sketch-only) | mcs4 (Sketch-only+Gate) | mcs5 (Raw-only) | mcs6 (Matte-CNN-only) |
|---|:---:|:---:|:---:|:---:|:---:|:---:|
| dense | 27.32 | 29.56 | 26.03 | 29.63 | **30.07** | 28.57 |
| mid | **29.32** | 27.03 | 23.67 | 27.83 | 28.88 | 27.20 |
| sparse | 17.67 | 14.22 | 17.49 | **20.18** | 19.79 | 14.99 |

방향성:
- **Sparsity ladder 효과**: Edge IoU 명확 감소 (0.045 → 0.027 → 0.010), Chroma도 감소.
- **design.md "sparse → mcs1 > mcs5" 예측 ✓ (v1과 반대로 v2에서 부분 성립)**: sparse Edge IoU mcs1(0.0096) > mcs5(0.0086) — 격차 0.0010 미세하지만 방향 일치.
- **design.md "Raw-only 색 탁(muddy)" 예측 ✗ (반대)**: sparse Chroma mcs5(19.79) > mcs1(17.67) — raw colored 입력에서는 mcs5가 matte raw로 색 더 잘 유지.
- **Chroma 절대값** v1보다 v2가 dense 영역에서 5~8 더 높음 (raw colored → 채도 그대로 전달, GT-recolor의 평균색 감쇠 효과 부재 확인).
- **Gate (mcs2) sparse Edge IoU 1위** (0.0127) — sparse 입력에서 gate가 sketch 정보 증폭에 기여, 단 Chroma는 mcs2(14.22)가 6모델 중 최저로 muddy 경향.
