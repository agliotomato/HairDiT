# [0718] Joint 재학습 결과 (GT / Colorful sketch)

기존 코드를 논문 방식대로 수정한 뒤 **joint 재학습**한 결과를 epoch별로 비교한다.
수정 내역은 [reports/[0713]training.md](%5B0713%5Dtraining.md) 참고.

## 개요

| 항목 | 내용 |
|------|------|
| 데이터 | 6,000장 (braid 3,000 + unbraid 3,000) |
| 학습 | phase1 30 epoch → phase2 20 epoch |
| seed | 42 고정 |
| 조건 비교 | **GT sketch** vs **Colorful sketch** 각각 추론 |
| 추론 시점 | phase1: e10 · e30 / phase2: e5 · e10 · e15 · e20 |
| 소스 이미지 | `data/paper`, `data/unbraid_new` 의 img · sketch(_gt) |
| 결과 이미지 | `outputs/results/joint_phase*_epoch*/` |

> 표 열: `img` (원본) · `sketch` (조건 입력) · `P1·eN` (phase1) · `P2·eN` (phase2)

## 결과

### 관찰
- **어두운 머리** 표현이 확실히 개선됨
- **phase2 epoch가 지날수록 결과가 나빠짐**: 색 반영이 약해지고, 머리 질감이 더 푸석해짐.

### 분석 — phase1 vs phase2 (loss 구성 차이)
phase1과 phase2의 차이는 loss 구성

| phase | loss |
|---|---|
| phase1 | `1.0·Flow + 0.1·LPIPS` — LPIPS는 학습 30% 이후 활성 |
| phase2 | `1.0·Flow + 0.1·LPIPS + 0.05·Edge` — LPIPS 즉시 활성 + **Edge loss 추가** |

### 이전 training 대비 loss 변경점
- **Flow loss 정규화**: 전체 픽셀수로 정규화 → **헤어 영역 픽셀수** 로 정규화
- **Edge loss stroke 판정**: threshold `0.1 → 1e-3` 으로 낮춤. max 채널 기준은 그대로지만 임계값이 거의 0이라, 사실상 **`(0,0,0)`(배경)이 아니면 stroke**로 판정.
